================================================================================
QR CODES - CE QU'ILS FONT
================================================================================

Le QR code contient:
  → Une URL complète: http://localhost:8000/register/?code=LIB4F6

Quand scanné:
  → Le navigateur mobile ouvre cette URL
  → Le formulaire d'inscription s'affiche
  → Le code "LIB4F6" est pré-rempli
  → L'élève voit "✓ Code pré-rempli"
  → Il remplit les 5 autres champs
  → Il s'inscrit

Avantages:
  ✓ Zéro erreur (code auto-rempli)
  ✓ Zéro copie (pas de copier-coller)
  ✓ Zéro authentification (direct du téléphone)
  ✓ Zéro effort (juste scanner)
  ✓ Traçabilité (code unique par librairie)
  ✓ Imprimable (image PNG simple)
  ✓ Gratuit (pas de service externe)

EXEMPLE CONCRET
================================================================================

Librairie du Centre (Code: LIB4F6)

1. JOUR 1 - Admin crée la librairie
   /admin/ → Librairies → Ajouter
   → Code auto-généré: LIB4F6

2. JOUR 1 - Admin dit à la librairie
   "Allez sur /register/qrcodes/ et téléchargez votre QR code"

3. JOUR 2 - Librairie télécharge et imprime
   /register/qrcodes/
   → Télécharge image du QR code
   → L'imprime en grand (20cm x 20cm)
   → Plastifie
   → Affiche sur le comptoir

4. JOUR 3 - Élève arrive
   "Bonjour, je voudrais m'inscrire à l'école"
   Librairie: "Scannez ce code avec votre téléphone"

5. ÉLÈVE SCANNE
   Téléphone → Scan QR
   Navigateur ouvre: /register/?code=LIB4F6
   Formulaire affiche: "Code pré-rempli: LIB4F6" ✓

6. ÉLÈVE REMPLIT 5 CHAMPS
   - Prénom: Ahmed
   - Nom: Benali
   - Email: ahmed@example.com
   - Téléphone: 0671234567
   - Niveau: 1ère année

7. ÉLÈVE CLIQUE "S'INSCRIRE"
   Formulaire vérifie
   Élève enregistré ✓
   Commission: 1500 DA générée pour la librairie ✓

8. ADMIN PAIE LA LIBRAIRIE
   /admin/partnerships/payment/
   → Ajouter paiement
   → Librairie: Librairie du Centre
   → Montant: 1500
   → Marquer comme complet

9. LIBRAIRIE VOIT LE PAIEMENT
   /dashboard/library/LIB4F6/
   → 1 élève inscrit
   → 1500 DA généré
   → 1500 DA payé
   → Statut: PAYÉ ✓

================================================================================
TECHNIQUEMENT
================================================================================

QR Code = Image PNG statique
  → Contient: http://localhost:8000/register/?code=LIB4F6
  → Scannable avec n'importe quel téléphone
  → Pas d'API, pas de serveur, pas de connexion
  → Juste une URL encodée en image

Processus:
  1. QR code généré avec: qrcode.QRCode()
  2. QR code affichée dans /register/qrcodes/
  3. Élève scanne → URL ouverte
  4. Paramètre ?code=LIB4F6 récupéré
  5. Formulaire pré-rempli avec get_initial()
  6. Élève remplit et soumet
  7. Normalement sauvegardé
