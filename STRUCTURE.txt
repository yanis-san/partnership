STRUCTURE SIMPLIFIÉE ET FONCTIONNELLE
=====================================

Page d'Accueil
└── http://localhost:8000/
    ├─ Description du système
    ├─ Lien vers Inscription
    ├─ Lien vers Dashboard Librairie
    └─ Lien vers Admin

Inscription Elève
└── http://localhost:8000/register/
    ├─ Formulaire d'inscription
    │  ├─ Prénom (requis)
    │  ├─ Nom (requis)
    │  ├─ Email (requis, unique)
    │  ├─ Téléphone (optionnel)
    │  ├─ Niveau (requis)
    │  └─ Code Partenaire (requis)
    │
    └─ Validation du code ✓
       └─ Code associé à une librairie

Confirmation d'Inscription
└── http://localhost:8000/register/success/
    ├─ Affiche les données de l'élève
    ├─ Affiche la librairie associée
    ├─ Affiche le code utilisé
    └─ Message de remerciement

Dashboard Librairie (Public)
└── http://localhost:8000/dashboard/library/[CODE]/
    ├─ Infos de la librairie
    ├─ Statistiques:
    │  ├─ Nombre d'élèves inscrits
    │  ├─ Commission par élève
    │  ├─ Montant généré
    │  ├─ Montant payé
    │  └─ Solde à recevoir
    │
    ├─ Statut de paiement (PAYÉ / PARTIEL / EN ATTENTE)
    ├─ Liste des élèves inscrits
    └─ Codes partenaires actifs

Dashboard Admin (Authentifié)
└── http://localhost:8000/dashboard/admin/
    ├─ Statistiques globales
    │  ├─ Nombre de librairies
    │  ├─ Nombre d'élèves
    │  ├─ Montant total généré
    │  └─ Montant total payé
    │
    ├─ Tableau des librairies
    │  └─ Avec toutes leurs stats
    │
    ├─ Paiements récents
    │  └─ Statut, montant, date
    │
    ├─ Élèves récents
    │  └─ Avec librairie associée
    │
    └─ Lien vers Admin Django

Admin Django Complet
└── http://localhost:8000/admin/
    ├─ Gestion des Librairies
    │  ├─ Créer
    │  ├─ Modifier
    │  ├─ Supprimer
    │  └─ Voir stats
    │
    ├─ Gestion des Codes
    │  ├─ Créer
    │  ├─ Modifier
    │  ├─ Activer/Désactiver
    │  └─ Voir usage
    │
    ├─ Gestion des Paiements
    │  ├─ Créer
    │  ├─ Modifier
    │  ├─ Marquer comme payé (action)
    │  └─ Voir historique
    │
    └─ Gestion des Élèves
       ├─ Lister
       ├─ Rechercher
       ├─ Filtrer
       └─ Voir détails

===== COMMANDES UTILES =====

Démarrer le serveur:
  python manage.py runserver

Accéder à l'accueil:
  http://localhost:8000/

Se connecter à l'admin:
  http://localhost:8000/admin/
  Username: admin
  Password: admin123

===== FLUX UTILISATEUR =====

1. Utilisateur arrive à l'accueil (/)
   ↓
2. Choisit une action:
   - Inscription (/register/)
   - Dashboard librairie (/dashboard/library/[CODE]/)
   - Admin (/admin/)
   ↓
3. Menu toujours visible en haut
   ↓
4. Navigation facile entre les pages

===== IMPORTANT =====

✓ Toutes les pages utilisent la même base (base.html)
✓ Même menu de navigation sur toutes les pages
✓ Formulaires simples et clairs
✓ Messages d'erreur affichés directement
✓ Codes en MAJUSCULES
✓ Emails uniques
✓ Commissions calculées automatiquement
✓ Pas besoin de CSS externe (inclus dans base.html)
